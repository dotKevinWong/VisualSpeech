# VisualSpeech
Helping the impaired speak
##
### Installation

0. Clone repoistory onto your machine using git 

1. Install Node.js and npm for your Local Environment
* Windows: [Node.js](https://nodejs.org/en/download/)
* macOS: ```brew install node```
* Debian: ```apt install nodejs```

2. Install Ionic CLI using npm
* ```npm install -g @ionic/cli```

3. Install npm dependencies
* ```npm install .```

4. Run App in Browser
* ```ionic serve```

5. Run App on Mobile
* You can follow the setup instructions for running the app on your mobile device(s). For [iOS](https://ionicframework.com/docs/developing/ios) devices, you will be required to use a Mac to install the app. For [Android](https://ionicframework.com/docs/developing/android) devices, you will need Android Studio.
